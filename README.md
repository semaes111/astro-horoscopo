# AstroDorado — Horóscopo Diario con IA

Dashboard freemium de horóscopos diarios generados por inteligencia artificial (Cassandra AI).

## Features

- 12 signos zodiacales con esculturas 3D doradas generadas por Gemini
- Horóscopos diarios generados por IA (estilo Co-Star español)
- 5 indicadores por signo: amor, fortuna, salud, trabajo, energía
- Detección automática de signo por fecha de nacimiento
- Card del signo del usuario pulsa con glow dorado
- Barra sticky de suscripción (Telegram / WhatsApp / Email)
- CTA personalizado según fecha de nacimiento
- Modal con ejemplo de lectura VIP (10 oráculos ancestrales)
- Efectos: Ethereal Shadow, Rainbow Borders, Vapour Text
- Responsive (4 → 2 → 1 columna)

## Stack

- HTML/CSS/JS puro (single file, ~1.8MB con imágenes base64)
- Imágenes: Gemini AI (esculturas zodiacales 3D doradas)
- Contenido: Cassandra AI (agente Paperclip)
- Efectos: SVG feTurbulence, conic-gradient, CSS animations
